SYSTEM: Estruture uma ROTINA DE DIETA com dias que se repetem.

ENTRADAS:
- Perfil do Usuário: {user_profile}
- Respostas do Questionário: {user_answers}
- Dias de dieta existentes no banco: {existing_diet_days}

REGRAS:
- Use dias existentes quando possível (liste-os em "reuse_days_by_name" se reaproveitar).
- "repetition_schema": "Semanal" | "Quinzenal" | "Mensal".
- Saída estrita em JSON. Sem texto fora do JSON.
- O número de placeholders na sequência pode variar conforme necessidade.

FORMATO:
{
  "diet_routine": {
    "name": "string",
    "description": "string",
    "repetition_schema": "Semanal|Quinzenal|Mensal",
    "day_sequence_placeholders": ["placeholder_1","placeholder_2"]
  },
  "days_to_create": [
    { "placeholder_id": "string", "name": "string", "description": "string" }
  ],
  "reuse_days_by_name": ["Dia já existente 1"]
}

EXEMPLO 1 (semana normal + fim de semana):
{
  "diet_routine": {
    "name": "Dieta Semanal com 2 livres",
    "description": "Semana padrão e 2 refeições livres no fim de semana.",
    "repetition_schema": "Semanal",
    "day_sequence_placeholders": ["semana","semana","semana","semana","semana","fim_semana","fim_semana"]
