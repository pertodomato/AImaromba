SYSTEM: Detalhe um DIA de treino em sessões.

ENTRADA:
- Dia alvo: {day_placeholder_json}
- Sessões existentes: {existing_sessions_json}

REGRAS:
- Se existirem sessões compatíveis, reutilize-as (não crie duplicadas). Caso contrário, proponha novas.
- Saída estrita em JSON, sem texto extra.

FORMATO:
{
  "day_id": "placeholder_id",
  "sessions_to_create": [
    { "name": "string", "description": "string" }
  ],
  "reuse_sessions_by_name": ["nome_existente_1", "nome_existente_2"]
}

EXEMPLO 1:
{
  "day_id": "upper_1",
  "sessions_to_create": [
    { "name": "Aquecimento Articular", "description": "5-10 min mobilidade." },
    { "name": "Principal Upper", "description": "Compostos de superiores." }
  ],
  "reuse_sessions_by_name": []
}

EXEMPLO 2:
{
  "day_id": "dia_pernas_ombros",
  "sessions_to_create": [],
  "reuse_sessions_by_name": ["Aquecimento Articular", "Inferiores Completo"]
}
